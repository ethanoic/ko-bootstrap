# ko-bootstrap
Knockout Bootstrap v4 

## Pre-requisites
1. NPM
2. Grunt CLI
3. Bower
4. SCSS
5. 
## How to setup
1. Clone this repo
2. Run "npm install" to install npm dependencies for build and testing
3. Run "bower install" to install libraries required for frontend

## How to ruN
Run "grunt server" and browse http://localhost:9002
